# CSMS Declaration
Firetruck Security Platform  
UNECE R155 – Cyber Security Management System (CSMS)

## 1. Manufacturer Information
- Manufacturer: Firetruck Security Platform (educational OEM)
- Scope: Firetruck Security Platform vehicle type and associated ECUs, gateways, and backend simulators
- CSMS Version: 1.0
- Date: __DATE__

## 2. Declaration of CSMS Implementation
The manufacturer declares that:

1. A Cyber Security Management System (CSMS) has been established, documented, and implemented in accordance with UNECE R155.
2. The CSMS covers the full lifecycle of the Firetruck Security Platform:
   - Development
   - Production
   - Post-production / in-service
3. The CSMS includes processes for:
   - Risk identification, assessment, and treatment
   - Cybersecurity testing and verification
   - Threat and vulnerability monitoring
   - Incident detection, response, and mitigation
   - Supplier cybersecurity management
4. The CSMS is applied to all cybersecurity-relevant assets, including:
   - Main ECU
   - Gateway ECUs (gw_chassis, gw_firecan, gw_hmi)
   - HMI ECU
   - CAN communication stack
   - Security monitoring system
   - OTA update subsystem (R156 linkage)

## 3. Commitment to Maintain the CSMS
The manufacturer commits to:

1. Maintain and continuously improve the CSMS.
2. Keep risk assessments, mitigations, and monitoring activities up to date.
3. Inform the Approval Authority of any significant changes to the CSMS that may affect type approvals.
4. Ensure that all future vehicle types under this CSMS are developed and managed in accordance with its processes.
