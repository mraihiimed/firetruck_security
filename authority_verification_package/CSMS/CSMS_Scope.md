# Cyber Security Management System (CSMS) – Scope

## 1. Vehicle Types Covered
The CSMS applies to the Firetruck Security Platform, including:
- Main ECU
- Gateway ECUs (gw_chassis, gw_firecan, gw_hmi)
- HMI ECU
- CAN communication stack
- OTA update subsystem
- Backend simulation components

## 2. Lifecycle Coverage
The CSMS covers:
- Development (software design, coding, testing)
- Production (build system, release packaging)
- Post‑production (monitoring, updates, incident handling)

## 3. Organizational Scope
Roles included:
- Cybersecurity Officer
- Software Developer
- Security Tester
- Release Manager
- Compliance Manager

## 4. System Boundaries
The CSMS includes:
- All ECUs and gateways
- CAN bus communication
- Security monitoring system
- OTA update mechanisms
- Logging and forensic data collection
