# Cyber Security Management System (CSMS)

## Scope (§7.2.2.1)
The manufacturer maintains a Cyber Security Management System covering:
- Development
- Production
- Post‑production
- Decommissioning

## Processes (§7.2.2.2)
- Cybersecurity governance
- Risk identification (Annex 5)
- Risk assessment & treatment
- Verification of mitigations
- Cybersecurity testing
- Lifecycle risk maintenance

## Monitoring (§7.2.2.2(g–h), §7.2.2.3)
- Threat & vulnerability monitoring
- Forensic data collection
- Timely mitigation of cyber threats

## Suppliers (§7.2.2.5)
Supplier dependencies are documented and monitored.

## Certificate (§6.x)
- CSMS documentation prepared
- Declaration signed
- Validity: 3 years
