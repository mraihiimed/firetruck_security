# SUMS Declaration  
Firetruck Security Platform  
UNECE R156 – Software Update Management System (SUMS)

## 1. Manufacturer Information

- Manufacturer: Firetruck Security Platform (educational OEM)
- Vehicle Type: Firetruck Security Platform
- SUMS Version: 1.0
- Date: 2026‑03‑25

## 2. Declaration of SUMS Implementation

The manufacturer declares that a Software Update Management System (SUMS) has been established, documented, and implemented in accordance with UNECE R156.

The SUMS covers:

- Software update planning
- Update creation and signing
- Update integrity verification
- Update distribution
- Update installation
- Rollback handling
- RXSWIN management
- Post‑update monitoring

The SUMS applies to all ECUs and gateways:

- main_ecu  
- gw_chassis  
- gw_firecan  
- gw_hmi  
- hmi_ecu  

## 3. Commitment to Maintain the SUMS

The manufacturer commits to:

1. Maintain and continuously improve the SUMS.
2. Ensure all updates are integrity‑protected and traceable.
3. Maintain accurate RXSWIN entries for all software versions.
4. Inform the Approval Authority of any significant changes to the SUMS.
