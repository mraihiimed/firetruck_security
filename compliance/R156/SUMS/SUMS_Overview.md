# SUMS Overview
Firetruck Security Platform  
UNECE R156 – Software Update Management System (SUMS)

## 1. Purpose
This document describes the Software Update Management System (SUMS) implemented for the Firetruck Security Platform in accordance with UNECE R156.

## 2. Scope
The SUMS covers:
- Main ECU
- Gateway ECUs (gw_chassis, gw_firecan, gw_hmi)
- HMI ECU
- OTA update subsystem
- RXSWIN register
- Update dossiers
- Configuration before/after snapshots
- Update logs and rollback logs

## 3. Objectives
The SUMS ensures:
- Secure software identification
- Integrity and authenticity of updates
- Safe update execution
- Traceability of all software versions
- Impact analysis before deployment
- Record retention for audits
