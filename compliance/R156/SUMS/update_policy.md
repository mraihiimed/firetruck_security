# Update Policy – Firetruck Security Platform  
**Version:** 1.0  
**Date:** 2026‑03‑24

## 1. Purpose
Define rules for distributing, validating, and applying software updates in compliance with UN R156.

## 2. Update Types
- Security patches  
- Feature updates  
- Certificate renewals  
- Logging/diagnostic improvements  

## 3. Deployment Rules
- Only signed packages may be installed.  
- Updates must pass integrity, safety, and power checks.  
- Rollback must remain available at all times.  
- User notification is mandatory unless update is classified as critical security.

## 4. Retention
All update records retained for **10 years**.
